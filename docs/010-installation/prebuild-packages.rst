Prebuild packages
=================

myMPD uses the openSUSE Build Service to build packages for releases.

- `Project homepage <https://build.opensuse.org/package/show/home:jcorporation/myMPD>`__
- `Repositories <https://download.opensuse.org/repositories/home:/jcorporation/>`__

Download the appropriated package for your distribution and install it with the package manager.

Following distributions are supported:

- Arch
- :doc:`Debian <prebuild-packages-debian>`
- Fedora
- Raspian
- openSUSE
- :doc:`Ubuntu <prebuild-packages-debian>`
