Distributions
=============

myMPD is available in the following distributions:

- `Alpine Linux <https://pkgs.alpinelinux.org/packages?name=mympd>`__
- `Arch Linux <https://archlinux.org/packages/?q=mympd>`__
- `DietPi <https://dietpi.com/>`__
- `GentooPlayer <http://gentooplayer.com/>`__
- `Guix <https://guix.gnu.org/>`__
- `IPFire <https://www.ipfire.org/docs/addons/mympd>`__
- `MusicLounge Audiophile Distro <https://linuxaudiofoundation.org>`__
- `MX Linux <https://mxlinux.org/>`__
- `NixOS <https://nixos.org/>`__
- `Snakeoil OS <https://www.snakeoil-os.net/>`__
- `Termux <https://termux.com/>`__

.. note::

   If you want to package myMPD for any other distribution don't hesitate to `contact me <https://github.com/jcorporation>`__.

myMPDos
-------

`myMPDos <https://github.com/jcorporation/myMPDos>`__ is my Raspberry Pi image (aarch64) based on Alpine Linux. It is running entirely in RAM and provides the latest MPD and myMPD packages and some goodies.
